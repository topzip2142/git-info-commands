# Основные комманды GIT <br>
---
###H3 Клонирование <br>
**git clone** — создаёт локальную копию удалённого репозитория. Указывается URL репозитория, и Git создаёт папку с именем репозитория (если не указать другое имя) и загружает туда всё необходимое.<br>
---
### Добавление изменений <br>
**git add** — добавляет файлы в индекс — специальную промежуточную область между рабочей директорией и репозиторием. В индекс можно добавить один файл, несколько или все сразу.<br>
*Некоторые варианты команды:*<br>
**git add имя_файла** — добавляет один указанный файл в индекс;<br>
**git add имя_файла1 имя_файла2** — добавляет несколько конкретных файлов;<br>
**git add .** — добавляет все изменённые и новые файлы в каталоге.<br>
**git --all** добавляет все файлы в папке проекта
---
### Отправка изменений в репозиторий <br>
**git commit** -m «Описание изменений» — формирует новый коммит с комментарием, описывающим суть внесённых правок.<br>
*Если не указать -m, то откроется консольный редактор vi, чтобы выйти - нажать **ESC**, потом ввести **:q!** - программа закроется*<br>
**git pull** - Отправляет изменения в репозиторий
