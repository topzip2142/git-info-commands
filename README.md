# Основные комманды GIT


### Клонирование

**git clone** — создаёт локальную копию удалённого репозитория. Указывается URL репозитория, и Git создаёт папку с именем репозитория (если не указать другое имя) и загружает туда всё необходимое.


### Добавление изменений

**git add** — добавляет файлы в индекс — специальную промежуточную область между рабочей директорией и репозиторием. В индекс можно добавить один файл, несколько или все сразу.<br>
*Некоторые варианты команды:
**git add имя_файла** — добавляет один указанный файл в индекс;
**git add имя_файла1 имя_файла2** — добавляет несколько конкретных файлов;
**git add .** — добавляет все изменённые и новые файлы в каталоге.
**git --all** добавляет все файлы в папке проекта


### Отправка изменений в репозиторий

**git commit** -m «Описание изменений» — формирует новый коммит с комментарием, описывающим суть внесённых правок.<br>
*Если не указать -m, то откроется консольный редактор vi, чтобы выйти - нажать **ESC**, потом ввести **:q!** - программа закроется*<br>
**git pull origin main** - Обновляет локальный репозиторий
**git push** - Отправляет изменения в удалённый репозиторий
